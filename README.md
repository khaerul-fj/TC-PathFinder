# 🚒 TC PathFinder: Mission-Critical Routing Engine

**TC PathFinder** adalah sistem navigasi cerdas yang dirancang untuk optimasi rute respons cepat menggunakan perpaduan algoritma **A* Search** dan **Dijkstra**.

### 🛠️ Fitur Utama:
* **Visualisasi Topologi:** Pemetaan jaringan dinamis dengan sistem label anti-tabrak.
* **Audit Data:** Transparansi penuh terhadap metadata JSON (Heuristic & Edge weights).
* **Efisiensi Tinggi:** Optimasi pencarian rute berbasis Priority Queue.

### 💻 Teknologi:
* Python (NetworkX, Matplotlib)
* Streamlit Cloud Deployment
